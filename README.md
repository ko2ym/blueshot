# Blueshot

Apple Silicon ネイティブ対応のmacOSスクリーンショットアプリです。

![Platform](https://img.shields.io/badge/platform-macOS%2013%2B-blue)
![Architecture](https://img.shields.io/badge/arch-Apple%20Silicon-green)
![Language](https://img.shields.io/badge/Swift-6-orange)

---

## 特徴

- **メニューバー常駐** — Dockに表示せず、常にメニューバーからアクセス可能
- **領域選択キャプチャ** — クロスヘアカーソル＋ルーペで精確な範囲指定
- **アノテーション** — 矢印・矩形・テキスト・ハイライト・ぼかし・モザイク
- **Undo/Redo** — 何度でもやり直し可能
- **グローバルホットキー** — アプリ非アクティブ時もショートカットキーで起動
- **PNG/JPEG出力** — ファイル保存またはクリップボードコピー
- **Apple Silicon ネイティブ** — M1/M2/M3/M4 チップに最適化

---

## 動作環境

- macOS 13 Ventura 以降
- Apple Silicon (arm64)

## インストール

### DMGからインストール（推奨）

1. [Blueshot_1.0.dmg](./Blueshot_1.0.dmg) をダウンロード
2. DMGをダブルクリックしてマウント
3. `Blueshot.app` を `Applications` フォルダにドラッグ

> **初回起動時の注意**
> Appleの公証（Notarization）未対応のため、Gatekeeperの警告が表示されます。
> Finderで `Blueshot.app` を右クリック →「開く」→「開く」をクリックしてください。
> 2回目以降は通常通り起動できます。

## 初回セットアップ

アプリ起動後、以下の権限を許可してください。

1. **画面収録** — スクリーンショット撮影に必須
   - 「システム設定」→「プライバシーとセキュリティ」→「画面収録とシステムオーディオ録音」→ Blueshot をオン

権限が未許可の場合、アプリ内に設定画面への誘導ボタンが表示されます。

## 使い方

### キャプチャ

メニューバーの Blueshot アイコンをクリックするか、ホットキーで起動します。

| 操作 | デフォルトのホットキー |
|------|-------------------|
| 領域選択キャプチャ | `Ctrl + Shift + 2` |
| ウィンドウキャプチャ | `Ctrl + Shift + 1` |
| 全画面キャプチャ | `Ctrl + Shift + 3` |

ホットキーは設定画面でカスタマイズできます。

### 領域選択

1. ホットキーまたはメニューから「領域選択キャプチャ」を選択
2. クロスヘアカーソルが表示されたらドラッグして範囲を指定
3. マウスボタンを離すとエディタが開く
4. `Esc` キーでキャンセル

### 編集（エディタ）

キャプチャ後にエディタが開きます。ツールバーから編集ツールを選択してください。

| ツール | 説明 |
|--------|------|
| 矢印 | 矢印を描画 |
| 矩形 | 四角形を描画 |
| テキスト | テキストを追加 |
| ハイライト | 半透明でマーキング |
| ぼかし | ガウシアンブラーをかける |
| モザイク | ピクセル化して難読化 |

- `Cmd + Z` — Undo
- `Cmd + Shift + Z` — Redo

### 保存

エディタ上部のボタンから出力先を選択します。

- **クリップボードにコピー** — 他アプリにそのまま貼り付け可能
- **ファイルに保存** — 設定した保存先フォルダにPNG/JPEGで保存

## 設定

メニューバーアイコン → 「設定...」から開きます。

### 一般
| 項目 | 説明 |
|------|------|
| ログイン時に自動起動 | Mac起動時にBlueshotを自動で起動 |
| キャプチャ後にエディタを開く | キャプチャ直後に編集画面を表示するか |
| マウスカーソルを含める | キャプチャ画像にカーソルを含めるか |
| キャプチャの遅延 | 撮影までの待機時間（0〜10秒） |
| 保存先 | クリップボード/フォルダ/両方 |
| 形式 | PNG / JPEG |
| JPEG 品質 | JPEGの圧縮品質（10%〜100%） |

### ファイル
| 項目 | 説明 |
|------|------|
| 保存フォルダ | ファイル保存先のフォルダ |
| ファイル名テンプレート | 変数を使ったファイル命名規則 |

**ファイル名テンプレートの変数**

| 変数 | 内容 |
|------|------|
| `${YYYY}` | 年（4桁） |
| `${MM}` | 月（2桁） |
| `${DD}` | 日（2桁） |
| `${hh}` | 時（2桁） |
| `${mm}` | 分（2桁） |
| `${ss}` | 秒（2桁） |

例: `Screenshot_${YYYY}${MM}${DD}_${hh}${mm}${ss}` → `Screenshot_20260327_153000.png`

### ホットキー
各キャプチャ操作のショートカットキーを変更できます。

## ビルド方法

### 必要環境

- Xcode 15 以降
- macOS 13 Ventura 以降

### ビルド手順

```bash
git clone https://github.com/ko2ym/blueshot.git
cd blueshot
xcodebuild -project Blueshot.xcodeproj -scheme Blueshot -configuration Release build
```
