# Blueshot 仕様書

バージョン: 0.1.0
最終更新: 2026-03-27

## 背景・目的

Greenshotというスクリーンショットツール（Windows/macOS対応）のmacOS Apple Silicon向け後継アプリを作成する。
GreenshotはバージョンアップがされておらずIntel Macでしか動作しない。Apple Silicon (M1/M2/M3/M4チップ) ネイティブ対応が必要。

## アーキテクチャ概要

```
Blueshot/
├── App/                        # エントリポイント・環境セットアップ
├── Domain/
│   ├── Models/                 # CaptureResult, HotKeyConfiguration 等
│   └── Errors/                 # BlueshotError
├── Features/
│   ├── Capture/                # CaptureCoordinator, ScreenCaptureService
│   │   └── RegionSelector/     # オーバーレイUI・ルーペ
│   ├── Editor/                 # エディタ画面（CanvasView, CanvasRenderer）
│   │   ├── Annotations/        # 各アノテーション実装
│   │   ├── Canvas/             # 描画エンジン
│   │   └── Tools/              # DrawingTool 定義
│   ├── Export/                 # ExportService, BookmarkManager
│   ├── HotKey/                 # Carbon API によるグローバルホットキー
│   ├── MenuBar/                # メニューバーUI
│   ├── Permissions/            # 権限チェック・誘導UI
│   └── Settings/               # 設定画面（一般・ファイル・ホットキー）
└── Infrastructure/
    ├── Logging/                # os.Logger ラッパー
    └── Persistence/            # AppPreferences (UserDefaults)
```
